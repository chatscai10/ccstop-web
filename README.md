# 輾好輾滿 · 網頁版

開改裝車輾殭屍的單機割草 Roguelike，純靜態 HTML5 Canvas，手機瀏覽器直接玩。原創作品。

線上試玩：https://chatscai10.github.io/ccstop-web/

## 玩法
- 虛擬搖桿開車輾殭屍，撿寶石升級三選一
- 8 台戰車、12 種武器、10 種被動、9 格裝備、30 關三大場景
- 救人質拿增益、打 Boss 過關
- 🛠 GM 面板：改資源/等級/裝備、無敵、一擊必殺、倍率調整
- 👑 GM 無雙關：全滿配 + 怪海模式

## 本地執行
```
python -m http.server 3210 --directory .
```
或直接開 index.html。
